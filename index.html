Diferença Cliente x Servidor

índice

O que são servidores

Um servidor é um computador especializado que armazena e gerencia um site, com informações como HTML, JavaScript, CSS, banco de dados e demais informações. O servidor envia e recebe as informações de acordo com as requisições do cliente, enviando ao navegador do usuário.

Cliente

O cliente envia solicitações para os servidores web pedindo informações, como abrir uma página da web. Após enviar essa solicitação, o servidor processa a requisição e envia de volta os arquivos necessários, como HTML, CSS, JavaScript e outros recursos, para que o navegador possa exibir o site corretamente.

O que é HTML?

índice

  • HTML significa Hyper Text Markup Language
  • HTML é o padrão de criação de páginas web
  • HTML descreve a estrutura de uma página web
  • HTML consiste de uma série de elementos chamados tags
  • HTML diz ao navegador como mostrar o conteúdo

Principais Tags HTML

índice

  • body - Contém todo o conteúdo visível na página, como texto, imagens, links, etc
  • head - Contém informações sobre o documento, como metadados, links para folhas de estilo e scripts, e o título da página.
  • h1, h2 ... h6 - Define os cabeçalhos, com h1 sendo o maior e h6 o menor
  • p - Define um parágrafo de texto
  • a - Cria um link para outra página ou recurso. O atributo href especifica o destino do link
  • div - Utilizada para agrupar e estruturar outros elementos HTML. É frequentemente usada com CSS para estilizar e formatar o conteúdo.
  • input - Cria campos interativos, como caixas de texto, botões e caixas de seleção. O tipo de input é especificado pelo atributo type
  • form - Define um formulário para coletar dados do usuário. Os elementos do formulário, como input, são colocados dentro desta tag.
  • ul, ol, li - Criam listas não ordenadas (ul), ordenadas (ol), e itens de lista (li), respectivamente.
